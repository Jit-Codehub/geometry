What is a Complementary Angle?

In mathematics, any two angles that add upto 90 degrees or π/2 is called complementary angle. The angles may not be adjacent to each other but when we add those angles, it should be 90 degrees.

For example, 60° + 30° = 90°.

Complementary Angle Formula

Look into the formulas, to find the complementary angle is given below.

Complementary angle = 90° - angle.

In Radians,

Complementary angle = π/2 - angle

  • Then, write the given angle with equal to 90 degrees as we know the definition of complementary angle. or use the formula of degrees or radians.

Complementary Angles Examples

Examples 1:

Question: Find the value of other angle if the complementary angle is 34°?

Solution:

Given, complementary angle is 34°

And we know the formula of complementary angle is

Complementary angle = 90° - angle

=> 90° - 34°

=> 56°

Therefore the other angle is 56°.

Example 2:

Question: Find the value of other angle if the complementary angle is 66°?

Solution:

Given, complementary angle is 66°

And we know that, When we add two angles, the sum should be equal to 90°.

i.e., 66° + x = 90°

Now, simplify the equation.

66° + x = 90°

X = 90° - 66°

=> 24°.

Therefore, the other angle x is 24°.
